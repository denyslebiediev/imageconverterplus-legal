개인정보 처리방침

사진은 당신의 것입니다. 모든 처리는 기기에서 이루어지며 — 업로드되거나 추적되거나 공유되지 않습니다.

사용자 기기에서

모든 이미지 변환, 크기 조정, 메타데이터 제거 및 워터마킹은 전적으로 사용자의 기기에서 이루어집니다. 원격 서버로 전송되는 것은 없습니다. 앱은 완전히 오프라인으로 작동합니다.

수집하지 않는 정보

당사는 개인정보, 사용 분석, 충돌 보고서, 기기 식별자 또는 기타 어떤 데이터도 수집하지 않습니다. 앱에는 추적, 원격 측정 또는 타사 분석 SDK가 없습니다.

사진 및 이미지

귀하의 사진과 이미지는 기기 외부로 업로드, 복사 또는 전송되지 않습니다. 앱은 귀하가 명시적으로 선택한 파일에만 접근하고, 로컬에서 처리한 후, 귀하가 선택한 위치에 결과를 저장합니다.

구매

앱 내 구매는 StoreKit을 통해 Apple이 전적으로 처리합니다. 당사는 결제 정보, 청구 정보 또는 Apple ID 데이터를 수신, 처리 또는 저장하지 않습니다.

공유 확장

공유 확장을 사용하여 앱에 이미지를 보낼 때, 해당 이미지는 기기에서 로컬로 처리됩니다. 외부 서비스에 업로드되거나 공유되지 않습니다.

데이터 보존

외부 서버가 존재하지 않으므로 외부 서버에 데이터를 저장하지 않습니다. 모든 변환된 파일과 설정은 귀하의 기기에 남아 귀하의 통제 하에 있습니다. 앱을 삭제하면 관련된 모든 데이터가 제거됩니다.

타사 서비스

앱이 통신하는 유일한 타사 서비스는 앱 내 구매 처리를 위한 Apple StoreKit입니다. 다른 외부 서비스, API 또는 SDK는 사용되지 않습니다.

개인정보 보호 매니페스트

앱에는 앱이 사용자 데이터를 수집하거나 추적하지 않음을 선언하는 Apple 필수 개인정보 보호 매니페스트가 포함되어 있습니다. 이 매니페스트는 App Store 심사 과정의 일부로 Apple이 검토합니다.
