Memulai

Ketuk tombol "+" di layar utama untuk memilih gambar dari Perpustakaan Foto atau aplikasi File. Anda juga dapat menggunakan Share Extension dari aplikasi apa pun — cukup pilih Image Converter Plus dari lembar berbagi.

Aplikasi mendukung JPEG, PNG, HEIC, TIFF, dan AVIF. Anda dapat mengonversi antara format-format ini.

Ya. Pilih beberapa gambar saat mengimpor dan semuanya akan dikonversi dengan pengaturan yang sama. Konversi massal tersedia dengan Pro.

Pengaturan Konversi

Gunakan slider kualitas di layar konversi. Nilai lebih rendah menghasilkan file lebih kecil, nilai lebih tinggi mempertahankan lebih banyak detail. Perkiraan ukuran file diperbarui secara real-time saat Anda menyesuaikan.

Ya. Anda dapat mengatur resolusi maksimum atau menskalakan gambar berdasarkan persentase. Rasio aspek selalu dijaga secara otomatis.

HEIC adalah format default Apple yang menawarkan kompresi lebih baik — file lebih kecil dengan kualitas serupa. JPEG kompatibel secara universal di semua perangkat dan platform. Konversi HEIC ke JPEG saat Anda perlu berbagi dengan perangkat non-Apple.

Privasi dan Metadata

Aktifkan "Hapus Metadata" di pengaturan konversi. Ini menghapus data EXIF termasuk lokasi, info kamera, dan cap waktu dari file yang dikonversi.

Ya. Ketika "Hapus Metadata" diaktifkan, semua data lokasi (GPS) dihapus bersama informasi EXIF lainnya. Ini berguna saat berbagi foto secara online.

Ya. Anda dapat menambahkan watermark teks dengan font, ukuran, warna, opasitas, dan posisi yang dapat disesuaikan. Watermark tersedia dengan Pro.

Menyimpan dan Berbagi

Anda dapat menyimpan gambar yang dikonversi ke Perpustakaan Foto, aplikasi File, atau membagikannya langsung ke aplikasi lain menggunakan lembar berbagi.

Dari aplikasi apa pun, ketuk tombol berbagi, lalu pilih Image Converter Plus. Gambar akan terbuka di Aplikasi siap untuk dikonversi. Semua pemrosesan tetap di perangkat Anda.

Ya. Setelah konversi, gunakan tombol berbagi untuk mengirim gambar ke aplikasi apa pun yang terinstal di perangkat Anda, termasuk media sosial, pesan, dan email.

Pro

Pro adalah pembelian sekali bayar yang menghapus semua batasan. Anda mendapatkan konversi tak terbatas, pemrosesan massal, semua format ekspor, watermark, dan semua pembaruan fitur di masa depan.

Ketuk banner Pro atau buka Pengaturan di dalam Aplikasi. Pembelian ditangani secara aman melalui App Store.

Buka Pengaturan di Aplikasi dan ketuk "Pulihkan Pembelian." Pastikan Anda masuk dengan Apple ID yang sama yang digunakan untuk pembelian asli.

Pemecahan Masalah

Buka Pengaturan > Privasi & Keamanan > Foto di perangkat Anda dan pastikan Image Converter Plus memiliki akses. Anda dapat memberikan "Akses Penuh" atau memilih foto tertentu.

Tidak ada batasan ukuran file tetap. Aplikasi dapat menangani gambar besar, tetapi file yang sangat besar mungkin memerlukan waktu lebih lama untuk diproses tergantung pada memori yang tersedia di perangkat Anda.

Coba tutup aplikasi lain untuk membebaskan memori, lalu coba konversi lagi. Jika masalah berlanjut, mulai ulang Aplikasi atau perangkat Anda. Anda juga dapat menghubungi kami di imageconverterplus.support@gmail.com.
